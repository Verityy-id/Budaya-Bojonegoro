Tari Kayangan Api

Budaya Lokal

Deskripsi Singkat

Tari Kayangan Api adalah tarian khas Bojonegoro yang terinspirasi dari fenomena alam Sumber Api Abadi di Desa Sendangharjo dan melambangkan semangat, kekuatan, serta kemakmuran masyarakat Bojonegoro. Tarian ini sering ditampilkan dengan kostum berwarna merah menyala dan gerakan lincah yang menyerupai api.

Seringkali menggunakan kain merah menyala, selendang kuning, dan elemen lain seperti kapas dan padi yang melambangkan kemakmuran.

Tema Koleksi

Gerakan lincah para penari menggambarkan keanggunan dan kekuatan seperti api abadi. Pola lantai dan properti seperti roda bergigi (usaha terus-menerus), rantai (persatuan), timbangan (keadilan), dan pohon beringin (simbol kehidupan) juga digunakan.

Makna dan Filosofi

Tari Kayangan Api tidak hanya sebuah pertunjukan seni, tetapi juga mengandung filosofi mendalam tentang semangat juang masyarakat Bojonegoro yang tidak pernah padam, seperti api abadi yang terus menyala. Tarian ini menjadi simbol kekuatan, keberanian, dan harapan yang terus hidup dalam setiap generasi.
